# AI 抠图 - MVP 需求文档

> 版本：v1.0 | 最后更新：2026-07-21

---

## 1. 产品概述

### 1.1 产品定位
在线 AI 图片背景去除工具。用户上传图片，一键去除背景，下载透明 PNG。

### 1.2 目标用户
- 电商卖家（商品白底图）
- 设计师（素材抠图）
- 普通用户（证件照换背景等）

### 1.3 核心价值
- **零门槛**：无需安装软件，浏览器打开即用
- **零成本**：纯前端部署，无服务器费用
- **隐私安全**：图片仅在浏览器内存处理，不上传

## 2. MVP 功能范围

### 2.1 本期包含（P0）

| 功能 | 说明 | 完成 |
|------|------|------|
| 图片上传 | 支持拖拽、点击选择，JPG/PNG/WebP | ✓ |

## 3. 技术方案

### 3.1 架构
```
用户浏览器
  ├── Vue 3 前端（静态资源）
  ├── remove.bg API（抠图推理）
  └── 内存处理（不落盘）
```

### 3.2 技术栈
| 层 | 选型 |
|------|------|
| 前端框架 | Vue 3 + Vite |
| 样式 | 原生 CSS（无第三方 UI 库） |
| AI 服务 | remove.bg API |
| 部署 | Cloudflare Pages |
| 存储 | 无持久化，仅浏览器内存 |

### 3.3 限制与约束
- 单张图片最大 **10MB**
- 依赖 remove.bg 服务可用性
- 免费账户每月 **50 张**
- 不支持离线使用

## 4. 用户流程

```
[进入页面] → [上传图片] → [等待处理] → [预览结果] → [下载 PNG]
                                                ↓
                                          [处理失败] → [错误提示] → [重新上传]
```

### 4.1 状态机
| 状态 | 触发条件 | 用户操作 |
|------|---------|---------|
| `upload` | 进入页面 / 重置 | 拖拽或点击上传 |
| `processing` | 文件选择完成 | 等待（无操作） |
| `result` | API 返回成功 | 下载 / 重新上传 |
| `error` | API 异常 / 校验失败 | 查看原因 / 重新上传 |

---

## 5. API 依赖

### 5.1 remove.bg
- 端点：`POST https://api.remove.bg/v1.0/removebg`
- 认证：`X-Api-Key` header
- 入参：`image_file`（multipart）、`size=auto`
- 出参：透明 PNG（binary blob）

### 5.2 错误码处理
| HTTP | 含义 | 用户提示 |
|------|------|---------|
| 200 | 成功 | 展示结果 |
| 402 | 额度用完 | "API 额度已用完，请检查账户" |
| 403 | Key 无效 | "API Key 无效，请检查配置" |
| 429 | 频率限制 | "请求过于频繁，请稍后再试" |
| 其他 | 未知错误 | "处理失败：{error}" |
